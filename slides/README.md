# Welcome to [Slidev](https://github.com/slidevjs/slidev)!

To start the slide show:

- `npm install`
- `npm run dev`
- visit <http://localhost:3030>

Edit the [slides.md](./slides.md) to see the changes.

Learn more about Slidev at the [documentation](https://sli.dev/).

## Guides

- [DEVELOPING.md](./DEVELOPING.md) — author and build the presentation locally.
- [UPDATING.md](./UPDATING.md) — publish changes to GitHub Pages.

## Deployment

These slides auto-deploy to GitHub Pages on every push to `master` that
touches `slides/**`. The workflow lives at
[`.github/workflows/slides.yml`](../.github/workflows/slides.yml). Manual
redeploy: GitHub → Actions → "Deploy slides" → Run workflow.

One-time setup: in the repo's Settings → Pages, set "Build and deployment"
source to **GitHub Actions**.
